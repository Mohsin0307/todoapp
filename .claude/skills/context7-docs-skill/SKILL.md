## When to Use
- When documentation is needed from Context7
- When MCP responses are long or repetitive
- When summarization is required

## Procedure
1. Query Context7 MCP with the topic
2. Extract relevant sections only
3. Remove examples unless requested
4. Summarize in bullet points

## Output Format
- Title
- Key Concepts (bullets)
- APIs / Functions (if any)
- Gotchas / Notes
